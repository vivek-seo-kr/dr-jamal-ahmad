# Dr. Jamal Ahmad | Cancer & Research Center

A clean, fast, and professional single-page portfolio website for **Dr. Jamal Ahmad**, Cancer Specialist and founder of the Doctor Jamal Ahmed Cancer & Research Center, Kangra, Himachal Pradesh.

🔗 **Live Demo:** [your-link-here.github.io](https://your-link-here.github.io)

---

## About

This portfolio is designed to help cancer patients and their families find and connect with Dr. Jamal Ahmad online. It highlights the center's expertise in painless cancer treatment through Naturopathy and advanced medicine therapy, making it easy for patients to book appointments and get directions to the clinic.

## Sections

- **Hero** — Introduction with specialization and call-to-action
- **About** — Doctor's background and center's philosophy
- **Services** — Cancer treatments and research specializations
- **Testimonials** — Patient success stories and reviews
- **Contact** — Location, phone, email and appointment booking
- **Map** — Embedded Google Maps for easy navigation

## Services Covered

- 🎗️ Cancer Treatment (All Stages)
- 💉 Naturopathy Injection Therapy
- 🔬 Research Center
- 🏥 Painless Treatment Programs

## Tech Stack

- Pure HTML, CSS & vanilla JavaScript
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Lucide Icons](https://unpkg.com/lucide@latest) for UI icons
- [Google Fonts](https://fonts.google.com/) for typography
- No frameworks · No build tools · No dependencies

## Deployment

This project is deployed via **GitHub Pages**.

To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`

## Local Preview

Just open `index.html` in any browser — no setup required.
